# Memory Model V3 — Demo Interativa

Demo interativa do [Memory Model V3](https://github.com/anthonystaiger8-bit/memory-model) — uma proposta original de arquitetura de memória para IA conversacional.

## 🚀 Testar agora

👉 **[anthonystaiger8-bit.github.io/memory-model-demo](https://anthonystaiger8-bit.github.io/memory-model-demo)**

Sem instalação. Sem chave de API. Abre direto no navegador.

## O que você vai ver

- 🧠 **Categorização MLP em tempo real** — cada mensagem é classificada automaticamente em Pessoa, Projeto, Ferramenta ou Ruído
- ⏱ **Timestamp por mensagem** — o sistema sabe exatamente quando cada informação chegou
- 📊 **Monitor de saturação de contexto** — visualize o contexto enchendo em tempo real
- ⚡ **Bypass Invisível (AEIL)** — quando o contexto satura, compressão automática é ativada
- 🌙 **Janela de Consolidação** — monitoramento do horário para consolidação silenciosa
